宝塔面板 搭建贪吃蛇 静态网站

一、准备
1. 工具：
宝塔面板
2. 环境：[宝塔环境]
LNMP

二、创建站点
1. 在宝塔页面添加站点
进入宝塔页面后，
点击：
网站——添加站点——输入域名——点击创建即可

2. 域名解析
	a. 进入阿里云，域名控制台
	b. 或在阿里云服务器控制台
		i. 添加解析

三、下载文件
1. 切换到，刚才创建的站点的目录
#
cd /www/wwwroot/snake.quansen88.cn/

2. 下载github中，已知可用的，贪吃蛇，静态网站源码
	【可下载到本地验证】
	#
	git clone https://github.com/yanhaijing/snake.git
	
3. 配置文件
方法一：
	将其中的文件，覆盖当前站点的数据文件。
	
或：
方法二：
	在宝塔面板中，将该站点的根目录，换为刚才下载的目录文件夹

四、验证是否成功

五、添加证书
若需要创建证书，参见相关教程。
已有证书可直接添加

可选择打开强制HTTPS
下一篇

docker生成免费Ssl 证书
