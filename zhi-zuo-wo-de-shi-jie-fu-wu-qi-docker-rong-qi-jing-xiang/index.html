一、vim 一个 Dockerfile 文件

FROM centos:7
RUN yum install wget java -y
#RUN mkdir /mc-server
#RUN cd /mc-server
RUN wget https://launcher.mojang.com/v1/objects/c5f6fb23c3876461d46ec380421e42b289789530/server.jar
RUN java -Xmx1024M -Xms1024M -jar server.jar nogui
RUN sed -i 's/eula=false/eula=true/g' eula.txt
CMD java -Xmx1024M -Xms1024M -jar /server.jar nogui

参考：http://quansen88.cn/post/zai-ben-di-da-jian-wo-de-shi-jie-fu-wu-qi-yi/

二、构建镜像

docker build -t mc-server:v0.2 .

三、运行容器

docker run --name mc-server -d -p 25565:25565 mc-server:v0.2
