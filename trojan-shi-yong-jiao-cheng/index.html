一、 服务器部署

1.1 docker安装mysql

[root@izt4n51g9tpto8eu51l4oyz ~]# docker run --name trojan-mariadb --restart=always -p 3306:3306 -v /home/mariadb:/var/lib/mysql -e MYSQL_ROOT_PASSWORD=trojanbf9ced6f983 -e MYSQL_ROOT_HOST=% -e MYSQL_DATABASE=trojan -d mariadb:10.2

1.2 安装Trojan

docker run -it -d --name trojan --net=host --restart=always --privileged jrohy/trojan init

1.3 进入Trojan容器

docker exec -it trojan bash

输入trojan安装

1. 选择 1.Let's Encrypt 证书
2. 输入域名
3. 数据库 直接回车
4. 数据库连接用户 直接回车
5. 数据库密码 输入1.1步骤的密码
6. 随机用户名 直接回车
7. 随机密码 直接回车

启动web服务

systemctl start trojan-web

设置开机启动

systemctl enable trojan-web
