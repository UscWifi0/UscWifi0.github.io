github-pages网页跳转（旧域名重定向到新域名）

2021-03-26

20200815-github-pages网页跳转（旧域名重定向到新域名）

原址：http://quansen88.cn/post/jiu-yu-ming-chong-ding-xiang-dao-xin-yu-ming/

一、创建并编辑 index.html

<script type="text/javascript">window.location.href="http://quansen88.cn";</script>

其中http://quansen88.cn是新域名

二、创建并编辑 文件CNAME

无后缀，大写，里面填上旧域名

三、将文件上传到github的 纯净博客仓库（用户名.github.io）

创建仓库 ---> 仓库名：（用户名.github.io）---> 上传文件

一般来讲需要重新注册一个github,专门用于此事

四、设置github pages

依次打开github----> 仓库_用户名.github.io -----> 设置（settings） -----> options (默认) -----> github pages ----->

Source -----> master -----> save (保存)

刷新后会显示旧域名， 这时仓库名作为网址已经可以重定向到新域名，并且允许旧域名解析
