docker/podman一键部署postwoman

2021-03-26

20200905docker/podman一键部署postwoman

官网：https://postwoman.io/

一、一键部署postwoman命令：

docker run --name pwm -d -p 3000:3000 liyasthomas/postwoman:latest

latest指的是dockerhub中该镜像的最新版本

二、查看是否成功

-p 3000:3000 那我们在网页中打开容器z所在ip的端口3000

#如我在本地安装的ip这样输入：
192.168.0.100:3000
