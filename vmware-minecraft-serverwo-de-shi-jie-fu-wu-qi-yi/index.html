一、安装与启动

下载服务器包时会给你一条命令

java -Xmx1024M -Xms1024M -jar minecraft_server.1.16.2.jar nogui

稍加思索

1. 安装java

yum install java

一般需要注意版本兼容，本次测试没有问题，java1.8

2. 对官网的命令稍加修改

java -Xmx1024M -Xms1024M -jar server.jar nogui

因为下载的文件，与命令中略微不同

然后会报错，不要慌，下一步

3. 根据提示修改配置文件eula.txt

vi eula.txt

编辑文件如下，即eula=flase改为eula=true

#By changing the setting below to TRUE you are indicating your agreement to our EULA (https://account.mojang.com/documents/minecraft_eula).
#Thu Aug 13 20:17:54 EDT 2020
eula=true

4.再次执行java

java -Xmx1024M -Xms1024M -jar server.jar nogui

5.未完待续，下一期：如何让客户端测试连接

脚本、命令历史

yum install java #v1.8，主要是与服务器包兼容
wget https://launcher.mojang.com/v1/objects/c5f6fb23c3876461d46ec380421e42b289789530/server.jar #服务器包
mkdir ~/mc-server #文件比较乱，创建文件夹放一起
mv ~/server.jar ~/mc-server/ #服务器包放进去
cd ~/mc-server/ #切换目录
java -Xmx1024M -Xms1024M -jar server.jar nogui #生成配置文件
sed -i 's/eula=false/eula=true/g' eula.txt #修改eula配置文件为true
java -Xmx1024M -Xms1024M -jar server.jar nogui #开始游戏
